# Edenara Group Website

Corporate website for **Edenara Group**, a diversified holding company operating across biotechnology and healthcare (Euphrora Medica), finance and investment (Gihora Capital), agriculture and food security (Pishora Agritech), and technology (Nexhora Technologies).

## Contents

- `index.html` — Single-file website (React + Babel, loaded via CDN). Open directly in a browser or deploy as a static site.

## Brand

- Palette: Navy (`#0B1F38`) and Gold (`#B8922A`)
- Typography: Cambria / Calibri

## Deployment

This is a static HTML file with no build step. To deploy:

- **GitHub Pages**: enable Pages for this repo (Settings → Pages → deploy from `main` branch, root folder).
- **Any static host**: upload `index.html` as-is (Netlify, Vercel, Cloudflare Pages, etc.).

## Notes

Requires an internet connection at runtime to load React, ReactDOM, and Babel from CDN.
